수강후기

  • 수강생 지민 님

    강의를 수강하기 전에는 UI/UX 프로세스를 정확하게 이해하지 못했던 부분들이 있었습니다. UX 직무와 UI 직무 둘 중에 선택하는 것도 어려웠고요. 강의를 듣고 난 지금은 UX와 UI, 두 가지 직무의 확연한 차이를 알게 되었고 직무를 제대로 선택하는 데 도움이 되었습니다.
    제대로 된 실무 프로세스를 알고 싶고, 실무 디자인을 할 때 어떤 포인트들을 제대로 알아야 하는지 궁금한 사람들에게 적합한 강의 라고 생각하고 만족스러워서 추천하고 싶습니다.

  • 수강생 용주 님

    강의를 듣기 전에는 어떤 디자인이 좋은 디자인인지 그 기준을 잘 몰랐습니다.
    지금도 잘 안다고는 말하기 어렵지만 좋은 디자인의 대략적인 기준은 알 것 같습니다. 제가 디자인 업무를 직접 하고 있지는 않지만 디자인 업무에 대한 이해가 필요했습니다. 강의를 수강하면서 인상 깊었던 점은 이 분야 에서 일하시는 디자이너 분들이 어떤 생각으 로 작업하시는지 알게 된 것이 아닐까 생각합 니다.
    도움이 많이 되었고, 좋은 선생님을 만나서 정말 보람찼습니다.

  • 수강생 상연 님

    저는 UX 디자이너를 꿈꾸는 수강생으로, 모바일 UI/UX 포트폴리오 강화가 필요해서 해당 강의를 수강하게 되었습니다.
    저는 강의를 듣기 전에는 수치 기반 디자인을 원했습니다. 그런데 강의를 듣고 난 지금은 디자이너의 눈, 감각도 그 이상으로 중요하단 걸 알았습니다.
    가장 기억에 남아 인상 깊었던 점은 강사님의 피피티가 알차서 정말 좋았습니다. UX를 이해하는 UI 디자이너가 되고 싶은 분들에게 이 강의를 추천해주고 싶습니다.
    이 강의를 들으면서 디자인적으로 많이 배울 수 있어 좋았습니다.

  • 수강생 연희 님

    강의를 수강하면서 '내가 체계적이지 않은 프로세스로 서비스를 만들었구나' 하는 생각이 들었습니다.
    지금은 하나씩 단계를 밟으며 프로젝트를 진행할 수 있게 되었습니다. 그동안 주먹구 구식으로 체계적이지 않은 프로세스로 작업을 해 온 사람들에게 추천합니다. 어디서 어떻게 시작해야 할지 막막할 때 작업을 시작할 수 있도록 도와줍니다.
    아주 사소한 질문이라도 강사님께서 친절하고 자세하게 설명해 주십니다. 수업을 들으면서 많은 것을 배웠고 덕분에 한 단계 더 성장하는 기회가 되었습니다. 감사합니다
